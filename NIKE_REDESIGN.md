# 耐克风格设计改造说明

## 改造概述

已将整个作品集网站按照耐克官网的设计语言进行了全面改造，打造出大胆、简洁、高冲击力的视觉体验。

## 核心设计变化

### 1. 配色方案
- **从**: 深色调（#070707）+ 金色强调（#d3a56d）
- **到**: 纯黑白配色（#000000 / #FFFFFF）
- 背景：纯黑或纯白，无渐变
- 强调色：黑白对比，无彩色点缀

### 2. 字体系统
- **从**: Inter + Noto Serif SC
- **到**: Helvetica Neue（耐克标准字体）
- 字重：更多使用 Black (900) 超粗字重
- 字号：更大更大胆（最大 8rem）
- 文字转换：大量使用 `text-transform: uppercase`

### 3. 布局风格
- **Hero 区域**:
  - 全屏沉浸式设计
  - 超大标题（4-9rem）
  - 极简文案
  - 黑底白字 CTA 按钮

- **作品展示**:
  - 从 12 列网格 → 2 列简洁网格
  - 无圆角设计（border-radius: 0）
  - 极窄间距（0.5-1rem）
  - 强烈的悬停缩放效果（scale 0.98 + 图片 1.08）

- **导航栏**:
  - 从玻璃态胶囊 → 全宽黑色顶栏
  - 简化 logo 尺寸（48px）
  - 大写字母导航
  - 白底黑字 CTA 按钮

### 4. 交互效果
- **悬停动画**: 更快更直接（200-350ms）
- **图片缩放**: 从 1.04x → 1.08x（更强烈）
- **卡片效果**: scale(0.98) 收缩效果
- **按钮**: 简单的 scale(1.02-1.05)

### 5. 间距系统
- 更大的留白空间
- padding: clamp(4rem, 8-12vw, 6-10rem)
- 区块间距增加到 5-12rem

## 设计特点对比

| 特性 | 原设计 | 耐克风格 |
|------|--------|----------|
| 配色 | 深灰+金色 | 纯黑白 |
| 字体 | Inter/Serif | Helvetica Neue |
| 圆角 | 1.5rem | 0 (无圆角) |
| 间距 | 中等 | 超大留白 |
| 字号 | 最大 7rem | 最大 9rem |
| 动画 | 柔和 | 快速直接 |
| 按钮 | 渐变/玻璃态 | 纯色对比 |
| 网格 | 12列复杂 | 2-3列简洁 |

## 耐克风格核心原则

1. **Just Do It 精神**: 简洁有力的文案
2. **运动感**: 快速的动画和强烈的视觉冲击
3. **极简主义**: 黑白配色，无多余装饰
4. **大胆排版**: 超大字体，全大写
5. **留白艺术**: 大量空白空间
6. **产品为王**: 图片占据主导，文字辅助
